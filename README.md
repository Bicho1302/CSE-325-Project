## Project Overview
StudyHub is a .NET Blazor web application designed to help students create, find, and manage study groups. The application provides an easy way for students to collaborate with classmates, join study groups, and organize study sessions.

## Features
- User Authentication
- Create Study Groups
- View Study Groups
- Edit Study Groups
- Delete Study Groups
- Search Study Groups by Course
- Join Study Groups
- View Study Sessions
- Input Validation and Error Handling

## Technologies Used
- .NET 10
- Blazor Server
- C#
- HTML
- CSS
- GitHub
- Trello
- Render

## How to Run the Application

1. Clone the repository:

```bash
git clone https://github.com/Bicho1302/CSE-325-Project.git
```

2. Navigate to the project folder:

```bash
cd CSE325Project
```

3. Run the application:

```bash
dotnet run
```

4. Open the local URL displayed in the terminal.
